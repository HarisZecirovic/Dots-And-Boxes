It is recommended to run it through pypy for better results:
Required steps:
pypy3 -m ensure pip,
pypy3 -m pip install django,
pypy3 -m install django_cors_headers,
then navigate to the folder where the project is located in the terminal and type the following:
ll_env\Scripts\activate,
pypy3 manage.py runserver,
and in the DotsandBoxesJavaScript folder, run the pocetna.html.
